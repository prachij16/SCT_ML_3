# SCT_ML_Task3 - Dogs vs Cats Classification using SVM

## Objective
Build an image classification model using Support Vector Machine (SVM) to classify images as Cats or Dogs.

## Dataset

The dataset used for this project is the Dogs vs Cats dataset from Kaggle.

Dataset files are not included in this repository due to GitHub file size limitations.

Dataset Link:
https://www.kaggle.com/c/dogs-vs-cats/data

## Tools Used
- Python
- Google Colab
- NumPy
- Matplotlib
- Scikit-learn
- SVM

## Steps Performed
1. Loaded image dataset
2. Resized images to 64x64
3. Converted images into numerical features
4. Created labels for cats and dogs
5. Split data into training and testing sets
6. Trained an SVM classifier
7. Evaluated model performance
8. Visualized predictions

## Result
Model Accuracy: 55%
